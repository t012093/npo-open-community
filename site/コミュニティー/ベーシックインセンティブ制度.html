    ベーシックインセンティブ制度¶

    概要

    ベーシックインセンティブ制度は、メンバーの貢献を適切に評価・還元し、持続的な成長を促進するための包括的な報酬システムです。

    1. 貢献度評価システム¶

    貢献度評価の特徴

    • 定量的な評価基準
    • 透明性の高い評価プロセス
    • 多面的な貢献機会の提供
    • 継続的な成長支援

    A[貢献活動] --> B[ポイント獲得]
    B --> C[ランク評価]
    C --> D[報酬付与]
    B --> E[実績の可視化]
    E --> F[認知向上]
    F --> A

    2. インセンティブの付与基準と獲得条件¶

    付与基準と獲得条件

    各インセンティブは、明確な基準に基づいて付与されます。

    インセンティブ 付与基準 獲得条件
    報奨金 プロジェクトの重要度、貢献度、成果の質 プロジェクトの重要な機能開発、重大なバグ修正、技術ドキュメントの拡充など、インパクトの大きい貢献をしたメンバー
    ギフト券 イベントへの参加頻度、コミュニティ運営への貢献度、アイデアソンの受賞など イベントへの積極的な参加、コミュニティ運営への貢献、アイデアソンでの受賞など、コミュニティへの積極的な貢献をしたメンバー
    コミュニティポイント コミュニティへの貢献度 (プロジェクトへの参加、イベント企画、情報提供など) プロジェクトへの参加、イベントの企画・運営、情報提供、コミュニティへの貢献活動を行うことで、ポイントが付与されます
    スキルアップ支援 コミュニティへの貢献度、スキルアップの必要性 コミュニティへの貢献度が高いメンバー、または、スキルアップが必要と認められたメンバー
    認知向上 コミュニティへの貢献度、成果の内容 プロジェクトで優れた成果を上げたメンバー、コミュニティに貢献したメンバー、技術ブログを執筆したメンバーなどを、ウェブサイトや成果報告書で紹介します
    キャリア支援 コミュニティへの貢献度、キャリアアップの意欲 提携企業からの推薦依頼があったメンバー、キャリアアップの意欲が高いメンバーに対して、キャリア支援を提供します

    3. インセンティブの金額と価値¶

    金額と価値の詳細

    各インセンティブは、貢献度に応じて適切な価値が設定されています。

    インセンティブ 金額/価値 備考
    報奨金 5,000円～50,000円 インパクトの大きい貢献に対しては、より高額な報奨金を検討
    ギフト券 500円～5,000円 Amazonギフト券または図書カード
    コミュニティポイント 1pt = 1円相当 イベント参加費割引（最大50%）、グッズ交換等に利用可
    スキルアップ支援 上限50,000円 研修費、書籍購入費の補助、メンター制度（無料）
    認知向上 - ウェブサイト掲載、表彰、ブログ執筆機会の提供
    キャリア支援 - 企業推薦、カウンセリング、インターンシップ機会

    4. ランク制度¶

    ランクと特典

    各ランクに応じて、受けられる特典が段階的に増加します。

    ランク 必要ポイント 特典
    ブロンズ 0-499pt 基本報酬、イベント参加費割引
    シルバー 500-999pt ブロンズ特典 + スキルアップ支援拡充
    ゴールド 1000-2999pt シルバー特典 + 技術顧問資格、高額報酬
    プラチナ 3000pt~ ゴールド特典 + 投資優先権、収益分配

    貢献ポイントシステム¶

    活動カテゴリ 獲得ポイント 評価基準
    プロジェクト開発 10-100pt/週 技術難易度、成果物の質、影響範囲
    イベント企画・運営 20-50pt/回 規模、参加者評価、運営品質
    技術ブログ執筆 10-30pt/記事 内容の質、反響、アクセス数
    技術支援・メンタリング 5-20pt/時間 支援内容、相手の成長度
    コミュニティ運営支援 5-15pt/回 貢献内容、効果
    ポイント獲得の具体例

    プロジェクト開発の例：

    • APIの新機能実装（中難度）: 50pt/週
    • バグ修正（軽度）: 10pt/週
    • アーキテクチャ設計（高難度）: 100pt/週
    • コードレビュー: 5-15pt/回
    • ドキュメント作成: 10-30pt/件

    よくある質問¶

    報酬はいつ支給されますか？

    基本報酬は四半期ごとに評価・支給されます。プロジェクト報酬は、成果達成時に随時支給されます。

    報酬額は固定ですか？

    基本報酬は固定ですが、プロジェクトの成果や特別貢献に応じて追加の報酬が付与されます。

    海外在住でも参加できますか？

    はい、参加可能です。報酬は国際送金で支払われ、各国の法令に従って適切に処理されます。

    6. NPO法人としての考え方¶

    重要な指針

    1. 非営利性の維持
      • 収益は活動とメンバーへ還元
      • 社会貢献を重視
    2. 透明性の確保
      • 収支の定期報告
      • 意思決定プロセスの公開
    3. 持続可能性
      • 適切な報酬水準の維持
      • 長期的な発展を考慮

    活動理念との調和¶

    graph LR
    A[社会貢献] --> B[適切な報酬]
    B --> C[持続的発展]
    C --> D[価値創造]
    D --> A

    制度の見直しについて

    本制度は、コミュニティの発展状況や社会情勢に応じて定期的に見直されます。変更がある場合は、十分な周知期間を設けて告知いたします。

    制度の目的

    このインセンティブ制度は、メンバーの成長と貢献を適切に評価・還元しながら、コミュニティ全体の持続的な発展を実現することを目的としています。
